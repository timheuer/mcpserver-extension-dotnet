# mcptest README

Sample with included .NET STDIO server

## Instructions

### Install global tool

```
dotnet tool install --global Mcp.Server.NuGet --version 0.1.0
```

### Run extension
- `npm install` (should get the experimental API definitions also)
- Open workspace in VSCode and F5/run
